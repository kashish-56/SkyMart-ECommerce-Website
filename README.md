# 🛒 SkyMart - Modern E-Commerce Website

SkyMart is a modern, responsive e-commerce web application built with **React**, **Tailwind CSS**, and **React Router**. It provides a seamless shopping experience with product browsing, filtering, searching, favorites, cart management, user authentication, and order management.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/kashish-56/skymart.git
```

Go inside the project

```bash
cd skymart
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```
